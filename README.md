# bun-library-tmpl

A template for bun library project.

## features

1. `commitlint`: lint commit message
2. `husky`: git hook
3. `@antfu/eslint-config`: eslint config
4. `release-it`: release tool
5. `happy-dom`: dom environment for testing

## usage

To install dependencies:

```bash
bun install
```

To lint your codes:

```bash
bun run lint:fix
```

To run test:

```bash
bun test
```

To build:

```bash
bun run build
```

To release:

```bash
bun run release
```
